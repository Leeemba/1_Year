# Дисциплина «Дискретная математика»

*В данном репозитории лежат материалы и все выполненные работы по дисциплине «Дискретная математика» за два семестра. Первый семестр закрывается зачетом,второй же экзаменом*

*В первом семестре дисциплина представлена домашним заданием в количестве восьми работ и курсовой,разбитой на две части*
*Во втором же семестре необходимо выполнить шесть домашних и работ и одну курсовую*
Также в обоих семестрах необходимо будет проходить текущее тестирование, основанное на контрольных работах
