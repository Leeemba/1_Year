# Дисциплина «Программирование» 

**В данном репозитории лежат выполненные лабораторные работы по программированию-дисциплине, преподаваемой в вузе «ИТМО»
Всего представлено 7 лабораторных,из который работы под порядковым номером 3 и 4 были объединены в одну для удобства.**

За первый семестр предстоит выполнить 4 лабораторные разной степени сложности.

За второй семестр предстоить выполнить 3 лабораторные обязательно и 1 дополнительно для получения высокой оценки.
