# Дисциплина «Программирование» 

**В данной директории лежат выполненные лабораторные работы по программированию-дисциплине, преподаваемой в вузе «ИТМО»
Всего представлено 7 лабораторных,из который работы под порядковым номером 3 и 4 были объединены в одну для удобства.**

За первый семестр предстоит выполнить 4 лабораторные разной степени сложности.

За второй семестр предстоить выполнить 3 лабораторные обязательно и 1 дополнительно для получения высокой оценки.
 

## First semester

- [1 лабораторная](./Lab1)  - основы языка, написания кода для решение математических уравнений
- [2 лабораторная](./lab2/Main.java) - основы ооп, написание кода при помощи сторонней библиотеки,способного запускать битву между командами покемонов  
- [3-4 лабораторная](./lab3/Main.java) - развитие изучения концепциии ооп, перевод текста сказки в логическую модель взаимодействия классов и их поведение
  
## Second semester

- [5 лабораторная](./lab5/main/java) - работа с коллекцией объектов в java,сериализация и десериализация данных в java, логика обработки данных в коллекции, правильное взаимодействие компонентов программы с сохранением принципа инкапсуляции  
- [6 лабораторная](./lab6/main/java) - изучение сетевого взаимодействия в java, создание приложения на двухуровневой клиент-серверной архитектуре по обработке коллекции объектов на основе логики из лабораторной работы №5 
- [7 лабораторная](./lab7/main/java) - подключение базы данных к java проекту, развитие двухуровневой архитектуры в трёхуровневую, внедрение механизмов многопоточной обработки данных
